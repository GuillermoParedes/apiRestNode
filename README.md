# Node Rest Starter Kit

> Api rest with NODEJS

## Requirements

* Nodejs
* Express
* MongoDB

## Clone Repo

```sh
$ git clone https://github.com/GuillermoParedes/apiRestNode.git
```

## Installation

```sh
$ cd apiRestNode
$ npm install
```

## Usage

```sh
$ npm start
```
